Player API

A JavaScript interface to control your videos and listen for events. Has universal support for both Flash and HTML5 player instances and a whole gang of convenience methods to make adding functionality easy.

Playlist API

Lets you control Playlist look and behavior at a high level. Provides hooks to give you Player API control over each video in the playlist.

Upload API

Programmatically upload videos to your Wistia account. For integrating with video-creation apps or adding as part of your video-exporting workflow!

Stats API

Access the raw analytics output for your account or specific medias. Also provides access to specific viewers and heatmap data.

Data API

For programmatic access to things in your Wistia account, like Projects Media, and Sharings (i.e. adding access for new users).

Embed API

Build the embed codes for your media programmatically - perfect for your dynamic content site!
